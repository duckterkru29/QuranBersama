# QuranBersama

Fitur Aplikasi:
1. Al-Quran Digital
114 Surah lengkap dengan terjemahan Bahasa Indonesia
Audio per ayat (klik tombol play)
Auto-track: Saat membaca surah, otomatis tercatat di Sijil Tsaksiyah bagian Tadarus
Tombol "Tandai Sudah Dibaca" untuk tracking manual per ayat
2. Hadits Shahih
Kumpulan hadits dari 9 sumber (Bukhari, Muslim, Tirmidzi, dll)
Teks Arab dan terjemahan
Fitur muat hadits acak
3. Jadwal Shalat
Otomatis berdasarkan kota (default: Jakarta)
Geolokasi untuk deteksi otomatis
Highlight waktu shalat saat ini
Kalender Hijriyah
4. Kegiatan Harian (Buku Amal)
Terbagi 3 kategori:
Ibadah: Shalat 5 waktu + Tahajjud, Tadarus (auto-track), Puasa
Olahraga: Lari, Push up, Angkat beban, Bela diri, Sit up, Futsal, Renang
Sosial: Silaturahmi, Kajian
Fitur Integrasi: Setiap kali membaca Al-Quran, aktivitas "Tadarus Quran & Terjemahan" otomatis ter-checklist.
5. Statistik
Progress bar harian
Skor per kategori
Statistik tadarus: Ayat dibaca, halaman setara, streak hari berturut-turut
Cara Mengakses Public:
Simpan kode di atas sebagai index.html
